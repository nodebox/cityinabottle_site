Getting started on Mac OS X + TextMate

nodebox.setup1.mov shows you how to start working with NodeBox for OpenGL if you have the TextMate editor. A test.py script is created in the same folder as the /nodebox folder (so we can import the library). The script is executed when CMD-R is pressed in TextMate. Easy!
